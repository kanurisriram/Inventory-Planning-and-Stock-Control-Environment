# Inventory Planning and Stock Control Environment

A modern **web-based business management dashboard** designed to help organizations manage inventory, customers, sales, purchases, expenses, and invoices efficiently.

This system provides a centralized platform to monitor business transactions, track financial activities, and maintain accurate inventory records.

---

# Project Overview

The **Inventory Planning and Stock Control Environment** simplifies business operations by integrating multiple management modules into a single dashboard.  

It enables businesses to monitor product inventory, track customer transactions, manage financial records, and generate invoices in a streamlined interface.

The system improves operational efficiency and reduces manual errors by maintaining structured records of all business activities.

# Installation and Setup

### 1 Clone the Repository

```
git clone https://github.com/yourusername/inventory-planning-system.git
```

### 2 Open the Project

Open the project in **IntelliJ IDEA** or any Java IDE.

### 3 Configure Database

Update database credentials in:

```
application.properties
```

### 4 Run Backend Server

Run the **Spring Boot application**.

### 5 Launch the Frontend

Open the frontend file in a browser:

```
index.html
```
